# 🏥 FED-EUS Benchmark Dataset  
**The first open, multi-center EUS image dataset for AI in the world!**

Hi! I am Dr.Qazi Aamir Suhail from India.  
I have a collection of EUS pictures from my hospital. This is the starting point (the "seed").

### What is this project?
Doctors take special ultrasound pictures inside the tummy (EUS).  
Right now no one in the world has a free, standard box of these pictures for AI to learn from.  
We are making one together!  

Doctors from India, Pakistan and all South Asia can add their pictures (we hide all patient names so it is 100% safe).  
Then anyone can use the box to train AI robots to find lumps, cysts or normal parts automatically.

### How to join (super easy)
- Read the rules (coming soon)  
- Get permission from your hospital  
- Send de-identified pictures with labels  
- You become a co-creator and get your name on papers!
